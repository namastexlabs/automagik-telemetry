## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/namastexlabs/automagik-telemetry.git
cd automagik-telemetry

# Python Development
cd python
pip install -e ".[dev]"
pytest --cov

# TypeScript Development
cd typescript
pnpm install
pnpm test
```
